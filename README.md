# Task Check

Uma agenda automatizada e de controle simples, desenvolvida em Python. Foco em produtividade com interface minimalista e automação invisível.

## Funcionalidades
### Funcionalidade Previstas

- **Adição automática de tarefas**: Digite a tarefa e ela é contada automaticamente.
- **Marcação de conclusão**: Marque com X para calcular o percentual de conclusão.
- **Gráfico de progresso semanal**: Visual simples que mostra o progresso (linha subindo para liberar dopamina).
- **Zero formatação**: Apenas escreva e marque X.
- **Interface burra por design**: Parece um bloco de notas simples.
- **Automação ao ligar o PC**: Janela abre automaticamente, salva tarefas e mostra gráfico.
- **Atalho no desktop**: Win + T abre diretamente o app.
- **Automação invisível**: Fórmulas e cálculos acontecem nos bastidores.

## Como Usar

1. Execute o app (`python app.py`).
2. Digite suas tarefas na interface.
3. Marque com X as concluídas.
4. Visualize o gráfico de progresso semanal.

## Requisitos

- Python 3.x
- Bibliotecas: tkinter, matplotlib (ou outras conforme necessário).
